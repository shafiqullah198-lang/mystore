# 🛍️ MyStore – Full Stack E-Commerce Web App

A modern full-stack e-commerce web application built with **Django**, featuring product browsing, advanced filtering, cart system, and a clean checkout experience.

---

## 🚀 Features

* 🛒 Product listing with filters (category, brand, price, rating)
* 🔍 Live search and sorting
* 🧾 Dynamic cart system (add/remove/update quantity)
* 💳 Modern checkout UI with payment form
* 📦 Order tracking system
* 🧑‍💼 Admin dashboard for product & order management
* ⚡ AJAX-based interactions (no page reloads)
* 🎨 Clean and responsive UI design

---

## 🖼️ Screens

* Product Listing Page
* Cart Page
* Checkout Page
* Admin Panel

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite (dev) / PostgreSQL (production)
* **Deployment:** Render / (Alternative: PythonAnywhere)

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/mystore.git
cd mystore
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

---

## 🔐 Environment Variables

Create a `.env` file or set variables:

```
SECRET_KEY=your_secret_key
DEBUG=True
ALLOWED_HOSTS=127.0.0.1,localhost
DATABASE_URL=your_database_url
```

---

## 📦 Deployment

1. Push code to GitHub
2. Connect repo to hosting (Render / PythonAnywhere)
3. Add environment variables
4. Run migrations
5. Deploy 🚀
